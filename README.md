# U.S. Consumer Behavior Under Inflation

## A Structural Break Analysis Using FRED Macroeconomic Data (2008–2026)

“Inflation surged in the post-COVID period, affecting prices, wages, interest rates, and household finances. But did this shock cause a temporary disruption, or has it fundamentally changed how Americans spend, save, and borrow?”

## 📊 Project Description

This project evaluates whether the post-COVID inflation shock created a temporary disruption or a permanent structural shift in U.S. consumer behavior.

This repository contains the Data Science / AI-ML / BIA track of the February Data Jam.

While the BI team focused on dashboards and storytelling, this analysis focuses on:
    - Structural break detection
    - Counterfactual forecasting
    - Residual diagnostics
    - Econometric hypothesis testing
    - Regime-based modeling
The goal was not just visualization but statistical proof.

## 📂 Data Sources

### Indicators
    - Real Personal Consumption Expenditures
    - Personal Saving Rate
    - Consumer Credit Outstanding
    - CPI (CPIAUCSL)

All data sourced from FRED.

## ▶️ How to Run the Project
    1. Clone Repository
    `git clone https://github.com/hamedtaeb/US-Consumer-Behaviour.git`
    `cd us-consumer-structural-break`

    2. Install Dependencies
    pip install -r requirements.txt

    3. Run Notebooks
    Open Jupyter:
    jupyter notebook

Run notebooks in order:
    1. Data Cleaning & Merge
    2. Feature Engineering
    3. Structural Break Analysis
    4. Prophet Counterfactual Modeling
    5. Hypothesis Testing
